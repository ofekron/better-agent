Executive summary

The dominant signal this week is the MCP 2026-07-28 specification release candidate — the largest protocol rewrite since launch, introducing a stateless core and a formal Extensions framework. It ships in days and is a breaking change for any MCP client, including Better Agent's connector layer. This is the highest-value item to act on.

Secondary opportunities cluster around MCP Registry discovery, Codex-style event Triggers / plugin parity, and MCP security hardening against the three new attack surfaces the stateless shift opens. Several orchestration patterns (LangGraph-style stateful graphs, role-based crews) are mature enough to borrow from.

One caution: a heavily hyped "OpenClaw" breakout GitHub project could not be independently corroborated from authoritative sources — it is held on the watchlist, not promoted to an opportunity.
